# 🧬 Panier de crise GFOFS
## Aliments à risque ultra-faible pour les phases de crise digestive sévère

Ce document liste **uniquement des aliments déjà présents dans les tables de classification GFOFS** qui présentent **des valeurs FAIBLES sur tous les marqueurs disponibles**.

Le Panier de crise est conçu pour les périodes de **forte instabilité digestive**, notamment :

- poussée d’IBS
- poussée de SIBO
- réactions à l’histamine
- récupération précoce du système digestif dans la maladie cœliaque
- instabilité digestive inexpliquée

L’objectif est **une prédictibilité digestive maximale avec un minimum de variables**.

# Critères de sélection

Les aliments inclus doivent respecter les critères suivants :

• Sans gluten  
• Faible en FODMAP  
• Aliment entier / peu transformé  
• **Faible en lectines**  
• **Faible en fibres (préféré)**  
• **Faible teneur en histamine**  
• **Faible interaction avec l’histamine**  
• **Faible charge lipidique**

Seuls les aliments déjà présents dans les **tables de classification GFOFS** sont pris en compte.

# Avertissement sur la couverture nutritionnelle

Le régime du Panier de crise est volontairement restrictif et doit être utilisé **temporairement**.

### Nutriments généralement suffisants

- Protéines  
- Vitamine B12  
- Fer  
- Sélénium  
- Potassium  

### Nutriments pouvant nécessiter une attention

- Vitamine C  
- Calcium  
- Magnésium  
- Zinc  
- Vitamine D  
- Oméga-3 (EPA/DHA)  
- Iode  
- Folates  

### Solutions pratiques

- Manger du poisson régulièrement ou envisager un complément d’huile de poisson pour les oméga-3
- Utiliser du **sel iodé** pour maintenir un apport en iode
- Consommer régulièrement papaye ou myrtilles pour la vitamine C ou envisager une supplémentation en vitamine C
- Envisager une supplémentation en magnésium, calcium et zinc
- Envisager une supplémentation en vitamine D
- Optionnel : faible dose de méthylfolate si l’apport en folates est faible

## Constipation / soutien de la motilité

Parce que le régime du Panier de crise est volontairement pauvre en fibres, certaines personnes peuvent souffrir de constipation.

Si une constipation apparaît, considérer les options douces suivantes :

Options préférées :

- supplémentation en magnésium (200–400 mg, le soir)
- psyllium (5–10 g/jour)
- hydratation adéquate
- marche régulière et mouvement

Aide naturelle optionnelle pour la motilité :

- kiwi (1–2 par jour si toléré)

Éviter les laxatifs stimulants tels que le séné ou le bisacodyl sauf indication médicale.

# Note importante d’utilisation

Le Panier de crise **n’est pas destiné à être un régime à long terme**.

Il est conçu pour :

1. Stabiliser la digestion
2. Réduire les déclencheurs inflammatoires
3. Simplifier les variables alimentaires

Une fois les symptômes améliorés, les aliments doivent être réintroduits progressivement à partir des **tables principales de classification GFOFS**.
